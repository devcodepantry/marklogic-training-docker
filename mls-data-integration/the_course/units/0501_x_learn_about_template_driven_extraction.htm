Exercise: Learn About Template Driven Extraction

In this walkthrough exercise, you access the documentation to learn about using TDE for generating rows, as well as, optionally, triples.

 

  1. In a browser window, navigate to docs.marklogic.com/guide/app-dev/TDE.
  2. Skim through the first few paragraphs, getting an overview of using TDE to extract row data from existing documents without modifying them.
  3. If desired, learn about TDE for triples by navigating to docs.marklogic.com/guide/semantics/tde.
  4. Skim through the topic, getting an overview of TDE for triples.
