Understanding the Data Hub Framework

The Data Hub Framework is an open source project that includes a graphical user interface for integrating data from silos.

The Data Hub Framework is available for download at github.com/marklogic/marklogic-data-hub.

The Data Hub Framework requires Java 8 (or later), and can be used with MarkLogic 8.0-7 (or later) or MarkLogic 9.0-1.1 (or later).

A step-by-step, hands-on tutorial is available at marklogic-community.github.io/marklogic-data-hub/tutorial.

Video tutorials on the Data Hub Framework are available at http://mlu.marklogic.com/ondemand/index.xqy?q=Operational+Data+Hub.

In addition, MarkLogic University offers a half-day hands-on workshop on using the Data Hub Framework.
