Exercise: Write a Model Descriptor

In this walkthrough exercise, you create a model descriptor for ProCycling's schema.

 

  1. On the desktop of your virtual machine, inside the mls-data-integration folder, create a new folder named es (right-click - Create Folder):


  2. Open the es folder.
  3. Create a new file inside the es folder (right-click - Create Document - Empty File), named procycling-model-descriptor.json.
  4. Open the procycling-model-descriptor.json file in a text editor.
  5. Enter the following, to define the descriptor (unit08/ex02a.txt):

    {
      "info": {
        "title": "procycling",
        "version": "0.0.1",
        "baseUri": "http://procycling.com/entity-types",
        "description": "ProCycling schema to define entity types, such as for products, orders, and customers."
      },
      "definitions": {

      }
    }
  6. Save the file.
