Creating an ODBC App Server

By creating an ODBC App Server for a MarkLogic database, external SQL queries can be made to the database through the defined port.

To create an ODBC App Server for a MarkLogic database, the Admin Interface or the REST API can be used.

The odbc server name field is simply a label, used internally.

The port field is for specifying the port number that the database will be available at.

The database field is for specifying which database is assigned to the port.
