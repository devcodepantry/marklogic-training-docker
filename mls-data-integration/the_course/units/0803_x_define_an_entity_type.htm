Exercise: Define an Entity Type

In this walkthrough exercise, you define a products entity.

 

  1. In the procycling-model-desc.json file, add a definition for an entity type named Product:

    {
      "info": {
        "title": "procycling",
        "version": "0.0.1",
        "baseUri": "http://procycling.com/entity-types",
        "description": "ProCycling schema to define entity types, such as for products, orders, and customers."
      },
      "definitions": {
        "Product": {

        }
      }
    }
  2. Add properties to the Product definition (unit08/ex03a.txt):

    {
      "info": {
        "title": "procycling",
        "version": "0.0.1",
        "baseUri": "http://procycling.com/entity-types",
        "description": "ProCycling schema to define entity types, such as for products, orders, and customers."
      },
      "definitions": {
        "Product": {
          "properties": {
            "id": {"datatype": "string"},
            "name": {"datatype": "string"},
            "price": {"datatype": "double"},
            "weightKg": {"datatype": "double"},
            "weightLb": {"datatype": "double"},
            "material": {"datatype": "string"},
            "gender": {"datatype": "string"},
            "color": {"datatype": "string"},
            "description": {"datatype": "string"},
            "speeds": {"datatype": "decimal"},
            "recommend": {
              "datatype": "array",
              "items": {"$ref": "#/definitions/Product"}
            }
          }
        }
      }
    }
  3. Optionally, define a primary key:

    {
      "info": {
        "title": "procycling",
        "version": "0.0.1",
        "baseUri": "http://procycling.com/entity-types",
        "description": "ProCycling schema to define entity types for products, orders, and customers."
      },
      "definitions": {
        "Product": {
          "properties": {
            "id": {"datatype": "string"},
            "name": {"datatype": "string"},
            "price": {"datatype": "double"},
            "weightKg": {"datatype": "double"},
            "weightLb": {"datatype": "double"},
            "material": {"datatype": "string"},
            "gender": {"datatype": "string"},
            "color": {"datatype": "string"},
            "description": {"datatype": "string"},
            "speeds": {"datatype": "decimal"},
            "recommend": {
              "datatype": "array",
              "items": {"$ref": "#/definitions/Product"}
            }
          },
              "primaryKey": "id"
            }
      }
    }
  4. Define required properties of name and price:

    {
      "info": {
        "title": "procycling",
        "version": "0.0.1",
        "baseUri": "http://procycling.com/entity-types",
        "description": "ProCycling schema to define entity types for products, orders, and customers."
      },
      "definitions": {
        "Product": {
          "properties": {
            "id": {"datatype": "string"},
            "name": {"datatype": "string"},
            "price": {"datatype": "double"},
            "weightKg": {"datatype": "double"},
            "weightLb": {"datatype": "double"},
            "material": {"datatype": "string"},
            "gender": {"datatype": "string"},
            "color": {"datatype": "string"},
            "description": {"datatype": "string"},
            "speeds": {"datatype": "decimal"},
            "recommend": {
              "datatype": "array",
              "items": {"$ref": "#/definitions/Product"}
            }
          },
              "primaryKey": "id",
          "required": ["name", "price"]
            }
      }
    }
  5. Save the file.
