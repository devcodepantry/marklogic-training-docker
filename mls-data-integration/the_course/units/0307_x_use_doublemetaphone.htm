Exercise: Use spell.doubleMetaphone()

In this walkthrough exercise, you use spell.doubleMetaphone to be able to match values that sound almost the same.

 

  1. In Query Console, in the workspace menu, select Import Workspace....
  2. Click the Browse button.
  3. Select Desktop - mls-data-integration - unit03 - spell.doubleMetaphone.xml.
  4. Click the Open button, and then the Import button.
  5. Select the first tab, spell:double-metaphone:

    spell:double-metaphone("Outback Rides")
  6. Click Run, to see the result:


  7. Select the second tab, Check for Equality:

    if (fn:deep-equal(spell:double-metaphone("Outback Rides"), spell:double-metaphone("At Back Rides")))
    then "match"
    else "no match"
  8. Notice the code to compare the double metaphone values of 'Outback Rides' and 'At Back Rides'.
  9. Click Run, to see the result:


  10. Select the third tab, Calls from Dee Monolakis:

    let $dm := spell:double-metaphone("Dee Monolakis")
    for $caller in /caller[fn:deep-equal(spell:double-metaphone(.), $dm)]
    return fn:doc(fn:base-uri($caller))
  11. Click Run, to see the result:

  12. See that both calls were returned, even though one had her name recorded incorrectly.
  13. If desired, navigate to www.marklogic.com/blog/master-data-management-justice-system, to read a blog post about using MarkLogic for master data management.
