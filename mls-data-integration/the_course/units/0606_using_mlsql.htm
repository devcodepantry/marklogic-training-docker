Using mlsql

Provided with MarkLogic on Linux and Windows, mlsql is a command line tool that is handy for testing external SQL queries.

Linux location: /opt/MarkLogic/bin/mlsql

Windows location: c:\Program Files\MarkLogic\mlsql.exe

The first step toward using mlsql is to connect it to the desired MarkLogic database, via the port that represents the ODBC app server for the database, as a user that is either admin or one with the sql-execution role.

/opt/MarkLogic/bin/mlsql -h localhost -p 3366 -U admin

Once mlsql is connected to a database, SQL queries can be entered at the mlsql prompt.


The end of queries in mlsql are designated with a semicolon (;).

To exit mlsql, \q can be entered.

Detail on mlsql is at docs.marklogic.com/guide/sql/setup#id_32009.

Tableau and Qlik

SQL queries can be made to a MarkLogic database from most SQL-based tools.

Steps for connecting Tableau to a MarkLogic database are at docs.marklogic.com/guide/sql/tableau.
