Exercise: Create an ODBC App Server

In this walkthrough exercise, you create an ODBC App Server at port 3366 for the customer360 database.

 

  1. In the Admin Interface, select Configure - Groups - Default - App Servers.
  2. Select the Create ODBC tab:


  3. For odbc server name, enter 3366-customer360-odbc.
  4. For root, enter /.
  5. For port, enter 3366.
  6. For database, select customer360.
  7. Click the ok button.
  8. Once it is done (i.e. the Waiting for localhost message in the lower-left corner is gone), click the Configure link in the upper-left corner.
  9. See the new ODBC App Server in the list:
