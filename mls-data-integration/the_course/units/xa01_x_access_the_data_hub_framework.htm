Exercise: Access the Data Hub Framework

In this walkthrough exercise, you access the Data Hub Framework and tutorial.


  1. In a browser window, navigate to github.com/marklogic-community/marklogic-data-hub.
  2. Realize that this is the GitHub landing page for the Data Hub Framework.
  3. Scroll down to the Getting Started section:

  4. Click the Getting Started Tutorial link.
  5. Skim through the text:


  6. Scroll down to the How Do I Get Started section:


  7. Click the Tutorial link.
  8. Skim through the tutorial's Prerequisites, and then Table of Contents:


  9. If desired, start doing the tutorial.
  10. If desired, view the on-demand Operational Data Hub video tutorial series at mlu.marklogic.com/ondemand/index.xqy?q=Operational+Data+Hub.
  11. If desired, view the MLU instructor-led training schedule at mlu.marklogic.com/registration, to see upcoming dates for the half-day workshop.
