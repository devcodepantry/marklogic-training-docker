Exercise: Get Set Up

In this walkthrough exercise, you open the course virtual machine and locate the course files.


  1. In VirtualBox, with the centos7_64_data_integration virtual machine selected, click the Start icon:



    Note: If you are using a 64-bit Windows computer but are unable to start the virtual machine running, it could be that Virtualization Technology is not turned on in the BIOS of your computer. The steps for turning it on vary, depending on the computer manufacturer. (For detail, use Google, or any search engine, to search on "enable virtualization technology".) For example, the sequence might be: restart the computer and hold down the F10 key while it is booting, then press Escape, then choose System Configuration, then Device Configurations, then Virtualization Technology, then change the value to enable, then Save.
  2. Wait for the virtual machine to boot.
  3. Click cent:



  4. For password, enter centos.
  5. Click the Log In button.
  6. See that you are now inside the virtual machine:


  7. On the desktop of the virtual machine, notice the mls-data-integration folder.
  8. Realize these are the course files.
  9. Double-click the folder, to open it:


  10. Double-click the folder named the_course.
  11. Double-click the file named index.htm, to open it in a browser.
  12. Realize that you are viewing the table of contents for the course:


(Optional) If desired, download the course files outside of the virtual machine, for easy access on your own computer:

  1. In a browser window on your own computer (outside the virtual machine), enter https://mlu.marklogic.com/downloads/ml9-data-integration-sp.zip to download a .zip file of the course files.
  2. Wait until the ml9-data-integration-sp.zip file has finished downloading.
  3. Unzip the file.
  4. See that you now have access to the class files (i.e. mls-data-integration folder) outside of the virtual machine.
