Exercise: Learn about SQL in MarkLogic

In this walkthrough exercise, you first access the MarkLogic SQL Data Modeling Guide, to learn about SQL in MarkLogic. Then you check that you have a template view defined, to use for SQL queries.

 

  1. In a browser, navigate to docs.marklogic.com/guide/sql.

  3. Select the SQL on MarkLogic Server link.
  4. Notice template view mentions:


  5. Click the browser's Back button, to return to the table of contents.
  6. Notice the Creating Template Views link, and select it.
  7. Skim the first paragraph:


  8. Click the browser's Back button, to return to the table of contents.
  9. Continue perusing, as desired.
  10. In Query Console, create a new workspace and name it SQL.
  11. For Database, select customer360.
  12. For Query Type, select XQuery.
  13. Enter the following, to see the view that you will be running SQL queries against, that you defined in the last unit:

    tde:get-view("customer360", "purchases")
  14. Press Run.
