Alternate Lab Steps: Create a Template

In this lab exercise, you create a template for the factsheets.

 

  1. In Query Console, view the factsheets documents, to see how they are structured.
  2. Note the names of the properties.
  3. Using XQuery or JavaScript, create a template that has a view name of factsheets, and columns named company_name, headquarters, phone, website, founded, employees, yearly_revenue, market, and notes.
  4. Have all of the columns be nullable except company_name.
  5. Validate the template.
  6. Insert the template into the customer360 database (to go into the customer360-schemas database in the background), giving it a URI of /templates/factsheet-template.xml.
