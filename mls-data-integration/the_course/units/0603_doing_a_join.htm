Doing a Join

Joins can be performed across views.

For example, imagine wanting to see purchases for each customer, from a purchases view, as well as market info for each customer, from a factsheets view.

SELECT purchases.company, purchases.product, purchases.quantity, factsheets.market
FROM purchases
INNER JOIN factsheets
ON purchases.company = factsheets.company_name
ORDER BY purchases.company

By referencing more than one view in the query, and connecting them through a common column value, a join can be performed.
