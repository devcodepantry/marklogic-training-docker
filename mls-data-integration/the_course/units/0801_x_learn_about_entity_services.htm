Exercise: Learn About Entity Services

In this walkthrough exercise, you access the Entity Services Developer's Guide.

 

  1. In a browser window, navigate to docs.marklogic.com/guide/entity-services:


  2. Select the Introduction to Entity Services link.
  3. Scroll down to the Why Use Entity Modeling section.
  4. Skim through the description:

  5. Continue to skim through the page, as desired.
  6. Click the browser's Back button, to return to the guide's table of contents.
  7. Click the Getting Started With Entity Services link.
  8. Skim through the page to get a high level view of using Entity Services, understanding that you can write your entity model descriptor in XML or JSON.

Load product documents:

  1. On the desktop of the virtual machine, open the mls-data-integration/unit08/data folder to see product documents:

  2. In a command prompt window, enter the following, to import the product documents into the customer360 database (unit08/ex01.txt):

    ~/Desktop/mlcp/bin/mlcp.sh import -mode local -host localhost -port 8055 -username admin -password admin -input_file_path ~/Desktop/mls-data-integration/unit08/data -output_uri_replace "home/cent/Desktop/mls-data-integration/unit08/data,'product'" -output_collections "raw"
  3. See that the command completes:

  4. In Query Console, select the customer360 database and click Explore.
  5. See that the documents loaded:
