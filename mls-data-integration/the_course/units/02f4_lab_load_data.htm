Lab: Load Data

In this lab exercise, you either use the steps below (normal steps), or the alternate steps (less detailed, so more challenging), to load the factsheet documents into the customer360 database.

 

  1. On the desktop of the virtual machine, in the mls-data-integration/unit02/data/sales-system/factsheets folder, right-click on one of the documents (e.g. outbackrides.json) to select Open With gedit, to open it in a text editor.
  2. See that the document contains factsheet data about the company:

  3. In a command prompt window within the virtual machine, run the following, to load the factsheets into the customer360 database (can copy from unit02/lab.txt):

    ~/Desktop/mlcp/bin/mlcp.sh import -host localhost -port 8055 -username admin -password admin -mode local -input_file_path ~/Desktop/mls-data-integration/unit02/data/sales-system/factsheets -output_uri_replace "/home/cent/Desktop/mls-data-integration/unit02/data/sales-system/factsheets,'/factsheet'" -output_collections "raw"
  4. In Query Console, with the customer360 database selected, click the Explore button.
  5. See that the factsheets loaded:

  6. Click the link for one of the documents (e.g. /factsheet/outbackrides.json) to see the contents:
